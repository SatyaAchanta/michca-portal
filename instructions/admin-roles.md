# Admin Roles

Introducing more Admin roles so that each admin have only access to specific pages.

## New Admin Roles

- UMPIRING_COMMITTEE ~ only can access Umpiring section of Admin
- WAIVER_COMMITEE ~ only can access Waiver section of Admin
- ADMIN ~ can access all three as of now and moving forward as well

### Instructions

- In Admin Section, all the different sections continued to be shown as usual, but before we display the actual content thats when we will display message if Admin is not allowed to see certain page. Message would be `You do not have enough permissions to see this page`
